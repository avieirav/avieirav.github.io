Mermaid to JSON Canvas Converter

Overview

This tool converts Mermaid flowchart diagrams into JSON Canvas format.

Features

  • ✅ Convert Mermaid flowcharts to JSON Canvas format
  • ✅ Support for TD (Top-Down) and LR (Left-Right) layouts
  • ✅ Automatic node positioning using dagre layout engine
  • ✅ Edge labels support
  • ✅ Color mapping from Mermaid styles to Canvas presets
  • ✅ Live preview with Mermaid rendering
  • ✅ Copy JSON to clipboard

Usage

  1. Enter Mermaid Code: Paste your Mermaid flowchart definition in the input textarea
  2. Select Layout Direction: Choose between TD (Top-Down) or LR (Left-Right)
  3. Convert: Click the Convert button to generate JSON Canvas
  4. Preview: View the rendered diagram in the preview section
  5. Copy JSON: Click “Copy JSON” to copy to clipboard
  6. Use Output: Paste the JSON into your application or save manually as needed

Example

Input (Mermaid):

graph TD
    A[Start] --> B{Decision}
    B -->|Yes| C[Action 1]
    B -->|No| D[Action 2]
    C --> E[End]
    D --> E

Supported Mermaid Features

Node Types

  • Rectangular: A[Label]
  • Decision/Rhombus: B{Label}
  • Rounded: C(Label)
  • Asymmetric: D>Label]

Connections

  • Basic: A --> B
  • With label: A -->|label| B
  • Thick: A ==> B
  • Dotted: A -.-> B

Styles

  • Inline colors: style A fill:#f96
  • Color mapping to Canvas presets (1-6)

Keyboard Shortcuts

  • Ctrl+Enter: Convert diagram

Technical Details

Dependencies

  • Mermaid.js v11: Parses and renders Mermaid diagrams
  • Dagre.js v2: Calculates automatic graph layout

Color Mapping

Mermaid ColorCanvas Preset

Limitations

  • Only flowchart diagrams are supported (not sequence, class, etc.)
  • Subgraphs are parsed but not rendered as groups in Canvas
  • Node dimensions are estimated based on text length
  • Complex styling may not fully translate

Troubleshooting

“Mermaid library not loaded”

  • Check your internet connection (CDN scripts need to load)
  • Try refreshing the page

“Invalid JSON output”

  • Check that your Mermaid syntax is valid
  • Use the preview to verify the diagram renders correctly
