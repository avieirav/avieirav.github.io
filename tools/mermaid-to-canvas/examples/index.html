Example Mermaid Diagrams for Testing

Basic Flowchart (Top-Down)

graph TD
    A[Start] --> B{Is it valid?}
    B -->|Yes| C[Process]
    B -->|No| D[Show Error]
    C --> E[End]
    D --> E

Flowchart with Colors (Left-Right)

graph LR
    A[Inicio] --> B{Decisión}
    B -->|Sí| C[Acción 1]
    B -->|No| D[Acción 2]
    C --> E[Fin]
    D --> E
    style C fill:#9f6
    style D fill:#f96

Complex Flowchart

graph TD
    A[User Login] --> B{Valid Credentials?}
    B -->|Yes| C[Dashboard]
    B -->|No| D[Error Message]
    C --> E{Logout?}
    E -->|Yes| F[Session Destroyed]
    E -->|No| C
    D --> A
    F --> A
    style C fill:#9f6
    style D fill:#f96
    style F fill:#69f

Testing Instructions

  1. Go to /tools/mermaid-to-canvas/
  2. Copy any of the examples above
  3. Paste into the Mermaid input textarea
  4. Select your preferred layout direction (TD or LR)
  5. Click Convert
  6. Preview the diagram and check the JSON output
  7. Click Copy JSON to copy the output
  8. Save as needed or use in your application
