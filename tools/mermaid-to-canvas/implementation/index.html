🧪 Testing

Manual Testing Required

  1. Browser Test

    • Open /tools/mermaid-to-canvas/
    • Verify page loads without errors
    • Check console for any JavaScript errors

  2. Conversion Test

    • Paste example Mermaid code
    • Click Convert
    • Verify JSON output is valid
    • Check preview renders correctly

  3. Layout Test

    • Test TD (Top-Down) layout
    • Test LR (Left-Right) layout
    • Verify both produce valid output

  4. Edge Cases

    • Empty input
    • Invalid Mermaid syntax
    • Complex diagrams with multiple edges
    • Diagrams with colors

🚀 Usage

For Users

  1. Navigate to /tools/mermaid-to-canvas/
  2. Paste Mermaid flowchart code
  3. Select layout direction (TD or LR)
  4. Click Convert
  5. Copy JSON to clipboard
  6. Save or use the output as needed

For Developers

# View the tool
open http://localhost:1313/tools/mermaid-to-canvas/

⚠️ Known Limitations

  1. Flowchart Only

    • Sequence diagrams not supported
    • Class diagrams not supported
    • Other Mermaid types not supported

  2. Layout Simplifications

    • Node dimensions are estimated
    • Complex layouts may need manual adjustment
    • Subgraphs parsed but not rendered as groups

  3. Style Mapping

    • Not all Mermaid styles translate to Canvas
    • Color mapping is approximate
    • Custom colors mapped to nearest preset
